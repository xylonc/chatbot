# Finance tracker chatbot 
"X" is a chatbot that will help your track your monthly finances (income, expenses , budgets...) with CLI!

# Motivation
X is a chat bot coded by myself using the concepts and skills I learnt in my univeristy's software engineering module. I wanted to use the knowledge I learnt and applied it to my very own project. Coincidentally my family and I were just about to head for an expensive holiday hence a financial tracker interested me, leading to me designing this chatbot
